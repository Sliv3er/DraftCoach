# Build DraftCoach Installer

This workflow produces a premium, branded Windows installer for DraftCoach.

## Architecture

The installer is a **2-layer system**:
1. **Tauri NSIS** (`apps/desktop-tauri/`) — builds the app and creates a silent NSIS installer
2. **Premium Wrapper** (`apps/installer/`) — Rust/WebView2 app with a 4-step wizard UI that embeds and runs the NSIS installer silently

The final output is a single `DraftCoach_Setup.exe` (~4 MB) that contains everything.

## Quick Build

// turbo-all

### Full build (Tauri + Wrapper):

```powershell
powershell -ExecutionPolicy Bypass -File scripts/build-installer.ps1
```

### Wrapper only (if Tauri/NSIS already built):

```powershell
powershell -ExecutionPolicy Bypass -File scripts/build-installer.ps1 -SkipTauri
```

### Tauri only (no wrapper):

```powershell
powershell -ExecutionPolicy Bypass -File scripts/build-installer.ps1 -SkipWrapper
```

## Output

```
dist/DraftCoach_Setup.exe          ← Final installer to distribute
apps/installer/target/release/     ← Build artifacts
```

## Prerequisites

- **Rust** toolchain installed (`rustup`)
- **Node.js 18+** for Tauri frontend build
- **WebView2 Runtime** on the build machine (comes with Windows 10/11)
- Dependencies installed: `cd apps/desktop-tauri && npm install`

## How It Works

1. `npx tauri build` compiles the Tauri app and creates `DraftCoach_1.1.0_x64-setup.exe` via NSIS
2. The wrapper's `build.rs` copies that NSIS exe to `embedded-setup.exe`
3. `include_bytes!` embeds the NSIS binary into the Rust wrapper at compile time
4. At runtime: user clicks Install → wrapper extracts NSIS to `%TEMP%` → runs it silently → cleans up

## Installer UX Flow

1. User double-clicks `DraftCoach_Setup.exe`
2. Premium borderless window opens (920×580, centered, dark+gold theme)
3. **Welcome** page shows features → click "Install Now"
4. **Location** page with install directory → click "Install"
5. **Installing** page with animated progress bar (NSIS runs silently in background)
6. **Finish** page with "Launch DraftCoach" button
7. Close button shows confirmation modal before cancelling

## File Structure

```

├── Cargo.toml          # Rust crate config
├── build.rs            # Copies NSIS exe + embeds icon
├── installer.rc        # Windows resource (icon)
├── icon.rc             # Generated by build.rs
├── src/main.rs         # WebView2 wrapper + IPC
└── ui/
    ├── index.html      # 4-page wizard template
    ├── styles.css      # Dark+gold premium CSS
    └── installer.js    # Page nav, progress, particles
```
